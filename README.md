# GA4 Demo Website

This repository contains a simple HTML demo website created for Google Analytics 4 (GA4) lab practice.

## Purpose
- To practice GA4 setup
- To analyze key metrics such as users, sessions, and engagement
- To implement custom events like button clicks and form submissions

## Files Included
- `home.html` – Demo website with GA4 tracking code and custom events
- `README.md` – Project description

## Tools Used
- Google Analytics 4
- HTML, CSS, JavaScript
- GitHub Pages

## How to Use
1. Open the live website using GitHub Pages
2. Perform actions such as clicking buttons and submitting forms
3. Check GA4 Realtime and Events reports
